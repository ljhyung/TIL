접근제한자 public, protected, (default), private가 있다.
public 모든 위치에서 접근이 가능
protected 같은 패키지에서 접근이 가능, 다른 패키지 접근 불가능, 단 다른 패키지의 클래스와
상속관계가 있을 경우 접근 가능
default 같은 패키지에서만 접근 허용, 접근 제한자가 선언이 안 되었을 경우 기본 적용
private 자신 클래스에서만 접근이 허용
