docker container 네트워크 설정

docker-compose.yml

```shell
version: '3'

services:
    jenkins:
        image: jenkins/jenkins:lts
        container_name: jenkins
        volumes:
            - /var/run/docker.sock:/var/run/docker.sock
            - /jenkins:/var/jenkins_home
        ports:
            - "8080:8080"
        privileged: true
        user: root
        environment:
            - TZ=Asia/Seoul

networks:
        default:
                external:
                        name: recipenav-net

```



이미 존재하는 네트워크로 설정해주기 위해 default:external:로 설정
